### Эффективность проекта

* t(ОЖ) Приложения = 10 мин; t(ОЖ) ПК = 30 мин

* Маркетолог имеет 20 желаний/сутки

* В приложении: 20х10=200 мин/сут; 200х20 = 4000 мин = 66,6 ч (за рассм. период 20 дней)

* Для ПК: 20х30 = 600 мин/сут; 600х20 = 12 000 мин = 200 ч

* Пусть в сутки пользуются 5 пользователей. Для прил.: 5х20х66,6 = 6660 ч/час; Для ПК: 5х20х20 000 ч/час

* 20 000 - 6660 = 13 340 ч/час/мес выгоды
